# java_practice_programs_without_OOPs
I'm creating this repo to help Java beginners, so that they can easy get access to "basic to advanced" practice question. Also this will help college/schools/institutes to get access to practice question for there students with solutions.

2 PDF are also uploaded in this repo which containing 250+ basic to mid difficulty questions.

Pls first try for solutions from your side.

I'll also upload 100+ mid to high level difficulty question in same approach. 

Note - All practive is without Java's OOPs concepts, will create a new repo for OOPs practice questions.
